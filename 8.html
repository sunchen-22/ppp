Image Transformation with Bilinear/Trilinear Filtering

Instructions:

  1. Upload an image using the button below
  2. Click on 3 points in the original image (red markers will appear)
  3. Click on 3 corresponding points in the result area (blue markers will appear)
  4. Click "Transform Image" to apply the transformation

The system will automatically use bilinear filtering when enlarging and trilinear when reducing.
